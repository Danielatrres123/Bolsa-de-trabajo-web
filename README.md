## Requisitos
- JDK 17+
- Maven
- MongoDB en localhost:27017

## Ejecutar backend
1. Abrir terminal en carpeta backend
2. Ejecutar: `mvn spring-boot:run`

## Ejecutar frontend
- Abrir `frontend/index.html` en el navegador o usar Live Server en VSCode.

## Endpoints principales
- GET /api/ofertas
- POST /api/ofertas
- POST /api/usuarios
- POST /api/postulaciones
